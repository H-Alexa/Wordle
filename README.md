

# Wordle-CLI

Wordle game with 2000 words to play in terminal.

## How to play

Enter the following command in the terminal:

```
git clone git@github.com:H-Alexa/Wordle.git
cd Wordle
python3 main.py
```

## Requirements

* Python3 interpreter
